Seguridad en redes con Evil Foca y Ettercap (ITSC)

Se evaluó la seguridad de redes universitarias mediante Evil Foca y Ettercap, explotando vulnerabilidades en la configuración de IPv6 con ataques SLAAC. Además, se utilizó Wireshark para capturar credenciales y analizar el tráfico de red. Con base en los resultados obtenidos, se implementaron mejoras en la configuración de red y en el uso de protocolos más seguros para mitigar riesgos.

Pruebas y validación de API con Postman (ITSC)

Se realizó API Testing utilizando Postman para verificar la funcionalidad, seguridad y rendimiento de los endpoints en un proyecto de desarrollo de API. Se llevaron a cabo pruebas de solicitudes HTTP (GET, POST, PUT, DELETE), validación de respuestas, manejo de autenticación y análisis de tiempos de respuesta. Además, se automatizaron pruebas con colecciones y scripts en JavaScript, asegurando la integridad de los datos y la correcta comunicación entre los sistemas que consumen la API.

Análisis Forense de Malware en Correos Electrónicos

Se desarrolló un proyecto de análisis forense para detectar y mitigar amenazas en correos electrónicos maliciosos. Utilizando herramientas como eml-analyzer, peedpf y VirusTotal, se identificaron indicadores de compromiso (IoC) en archivos adjuntos y enlaces sospechosos. El análisis reveló un PDF malicioso y una URL de phishing, además de confirmar que la cuenta del destinatario había sido expuesta en filtraciones previas. Como resultado, se propusieron medidas de seguridad para reforzar la protección contra ataques de phishing y malware.
